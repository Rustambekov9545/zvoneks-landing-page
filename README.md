# Звонекс

Интерактивный лендинг сервиса автоматизации холодных продаж.

## Возможности демо

- звонки из браузера;
- скрипты и работа с возражениями;
- настройка KPI и отчёты по менеджерам;
- автоматические задачи после пропущенных звонков;
- AI-анализ разговоров с голосовым воспроизведением.

Сайт построен на HTML, CSS и JavaScript без сборщика.
